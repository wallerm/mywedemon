### BackTop Props

| 名称  | 类型    | 默认值    | 说明                                                                | 必传 |
| ----- | ------- | --------- | ------------------------------------------------------------------- | ---- |
| fixed | Boolean | true      | 是否绝对定位固定到屏幕右下方                                        | N    |
| icon  | String  | 'backtop' | 图标                                                                | N    |
| show  | Boolean | false     | 是否显示组件                                                        | N    |
| text  | String  | ''        | 文案                                                                | N    |
| type  | String  | round     | 预设的样式类型。可选值：round/half-round/round-dark/half-round-dark | N    |

### BackTop Events

| 名称   | 参数 | 描述     |
| ------ | ---- | -------- |
| to-top | -    | 点击触发 |
